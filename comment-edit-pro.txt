== Description ==

Comment Edit Pro provides an options panel for Simple Comment Editing with a myriad of options including adjusting the timer, selecting a theme, and edit logging.

> Comment Edit Pro has been rebranded to Comment Edit Pro. If you have purchased a license, I will create a new free license for you. Please email me at ronald@dlxplugins.com or use <a href="https://dlxplugins.com/support/">my contact form</a> for this request.

Comment Edit Pro features:
<ol>
<li>Set the timer to a custom value and even hide it</li>
<li>Select a theme</li>
<li>Receive an email each time a user edits or deletes their comment</li>
<li>Set a minimum comment length when editing<li>
<li>Log edits so you can see an edit history when viewing the comment.</li>
<li>Edit button text and other messages.</li>
</ol>

== Installation ==

1. Just unzip and upload the "comment-edit-pro" folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
