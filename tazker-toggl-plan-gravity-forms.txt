Use Toggl Plan and Gravity Forms to streamline your task workflow.	

== Description ==

Use Gravity Forms and the Tazker add-on to automatically create tasks in Toggl Plan.

More information about Tazker: https://dlxplugins.com/tazker/

== Installation ==

1. Just unzip and upload the "tazker-for-toggl-plan" folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
